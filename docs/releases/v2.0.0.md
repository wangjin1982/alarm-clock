# 桌面闹钟 v2.0.0

## 本次更新

- 主界面升级为“极简模式”，移除顶部时间和日期展示，减少视觉干扰。
- 顶栏取消九宫格布局编辑入口，保留更克制的窗口控制、Logo 与设置按钮。
- `Jinger` Logo 调整到顶栏右侧，并缩小到更适合当前头部空间的尺寸。
- 番茄钟卡片重新排版，主时间移动到右侧显示，层级更清晰。
- 金句区域拆分为独立卡片，放置在番茄钟与闹钟之间，让整体呼吸感更好。
- 番茄钟时间上方仅保留闪烁状态点，去掉“进行中”等辅助文案，保持画面简洁。

## 升级说明

- 这是一次以界面整理为主的 2.0 升级，不涉及已有闹钟、番茄钟和提醒逻辑的破坏性改动。
- 本地设置会继续沿用，已有昵称、通知、声音和置顶设置不会丢失。
- 之前实验性的布局编辑模式已移除；当前版本重新回到固定布局，以保证界面统一和稳定。

## 当前主界面

- 极简顶栏
- 番茄时间管理
- 独立金句卡片
- 独立闹钟列表

## 备注

- 桌面提醒继续由 Rust 后台触发，应用不在前台时也能播报。
