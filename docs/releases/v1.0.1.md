# 桌面闹钟 v1.0.1

## 本次更新

- 顶栏时间与日期改成单行显示，整体观感更整洁。
- 调整窗口初始高度，减少主界面底部空白。
- 番茄卡片右上角替换为 `Jinger` 品牌 Logo。
- 移除顶部昵称展示和锁定按钮，顶栏更简洁。
- 将闹钟从天气模块中拆出，主界面恢复为“番茄钟 + 闹钟”双卡片布局。
- 隐藏天气展示界面，但天气与定位相关代码仍保留在仓库中，便于后续继续开发。
- 闹钟卡片移除定位说明区域，并进一步压紧列表高度。
- 当前版本不再主动请求系统定位权限。

## 当前主界面

- 番茄时间管理
- 独立闹钟列表
- 顶栏时间日期与窗口控制

## 备注

- 桌面提醒继续由 Rust 后台触发，应用不在前台时也能播报。
